# AI Role Discussion Platform

A Vue 3 application that enables two AI roles (hero and villain) to discuss topics using the Mistral API.

## Features
- Customizable AI role prompts
- Configurable discussion rounds
- Automatic summary generation
- Local conversation history
- Responsive design

## Setup
See [USAGE.md](docs/USAGE.md) for detailed setup and usage instructions.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
